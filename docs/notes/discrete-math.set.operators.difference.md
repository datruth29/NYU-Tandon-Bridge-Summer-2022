The __difference__ between two sets $A$ and $B$, denoted $A - B$, is the set of elements that are in $A$ but not in $B$.

Example:

$$
A = \{a, b, c, e, f\}
B = \{d, e, f, g\}
A = \{a, b, c\}
$$
